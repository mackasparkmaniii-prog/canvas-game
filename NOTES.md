# Canvas Game Notes

## 1) Goal / Roadmap
- Build a simple canvas “game site” with 3 games:
  1. Rock Paper Scissors (Best of 3)
  2. Connect 5
  3. Checkers
- Start with a menu scene and reusable game framework.

## 2) Folder Structure
canvas-game/
- index.html
- script.js
- assets/
- NOTES.md

## 3) How to Run
- Use Live Server:
  - Right click index.html → Open with Live Server
- The game opens in the browser at:
  - http://127.0.0.1:5500/index.html
